## General 

| Command | Description |
| --- | --- |
| `⌘ + C` | Copy |
| `⌘ + X` | Cut to clipboard |
| `⌘ + V` | Paste from clipboard |
| `⌘ + C` -> `⌘ + option + V` | Copy and Paste |

## Chrome 
| Command | Description |
|--- | --- |
| `⌘ + n` | Open New Window | 
| `⌘ + l` | Go to browser bar |
